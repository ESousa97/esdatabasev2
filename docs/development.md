# Development Guide

> This project is deactivated and no longer under active development.
> The commands below are kept only for local/historical usage.

## Setup

```bash
npm install
npm run dev
```

## Linting & Formatting

```bash
npm run lint
npm run format:check
```

## Tests

```bash
npm run test
npm run test:coverage
```

## Build

```bash
npm run build
npm run start
```

## Git Hooks

Husky runs lint + format on staged files before commit.
