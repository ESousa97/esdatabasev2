# Security Policy

## Supported Versions

We actively support the latest major version of this project.

| Version | Supported |
|---------|-----------|
| 2.x     | ✅ Yes     |
| < 2.0   | ❌ No      |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

- Email: [security@esdatabase.dev](mailto:security@esdatabase.dev)
- Provide a clear description of the issue
- Include steps to reproduce
- Include impact analysis if possible

We will respond within **72 hours** and provide a remediation timeline.

## Responsible Disclosure

We appreciate responsible disclosure and will credit reporters if appropriate.

---

Thank you for helping keep this project secure.
