# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-02-03

### Added

- Centralized design tokens and updated theme configuration
- ESLint, Prettier, Jest and Testing Library setup
- GitHub Actions CI pipeline and Dependabot configuration
- Security policy, contributing guide and code of conduct
- Husky + lint-staged for pre-commit validation

### Changed

- Rewrote README with concise architecture and usage
- Updated NextAuth configuration to use environment-based allowlist
- Improved global styles and accessibility baseline

### Removed

- Deprecated duplicate utils file
